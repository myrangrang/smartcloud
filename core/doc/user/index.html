smartCloud 1.0 User Manual Introduction¶

Welcome to smartCloud: your self-hosted file sync and share solution.

smartCloud is open source file sync and share software for everyone from individuals operating the free smartCloud Server edition, to large enterprises and service providers operating the smartCloud Enterprise Subscription. smartCloud provides a safe, secure, and compliant file synchronization and sharing solution on servers that you control.

You can share one or more files and folders on your computer, and synchronize them with your smartCloud server. Place files in your local shared directories, and those files are immediately synchronized to the server and to other devices using the smartCloud Desktop Sync Client, Android app, or iOS app. To learn more about the smartCloud desktop and mobile clients, please refer to their respective manuals:

  • smartCloud Desktop Client
  • smartCloud Android App
  • smartCloud iOS App
